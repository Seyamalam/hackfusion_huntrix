# Slidev Deck

This folder contains the final pitch deck source in Slidev format.

## Run

```bash
cd docs/slidev
npm install
npm run dev
```

Default local URL:
- `http://localhost:3030`

## Export

```bash
cd docs/slidev
npm install
npm run build
npm run export
```
